# <a name="sorting"></a>Sortieren
Dank der Sortierfunktion können Sie sich schnell und einfach einen Überblick über Ihre Daten verschaffen. Wenn Sie beispielsweise über zahlreiche Debitorenkontakte verfügen, können Sie auswählen, um nach folgenden Kriterien zu sortieren: **Debitorennummer**, **Debitorenbuchungsgruppe**, **Währungscode**, **Länder-/Regionscode** oder **Salestax-Registrierung**. zu eine Übersicht zu erhalten, die Sie benötigen.

Wählen Sie eine Spaltenüberschrift und verwenden Sie den kleinen Pfeil, um aufsteigend oder absteigend zu sortieren.  

> [!NOTE]  
>   Sortierung wird nicht auf Bilder, BLOB-Felder und FlowFilter unterstützt, die keiner Tabelle angehören.
