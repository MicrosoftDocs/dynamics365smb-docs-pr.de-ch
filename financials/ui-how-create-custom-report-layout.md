1. <span data-ttu-id="47aee-115">Wählen Sie das Symbol ![Nach Seite oder Bericht suchen](media/ui-search/search_small.png "Search for Page or Report icon") und geben **Layout-Berichtsauswahl** ein.    <span data-ttu-id="47aee-116">In dem Fenster **Bericht-Layioutabschnitt** sind alle Berichte aufgelistet, die für das Unternehmen verfügbar sind, das im Fenster Bericht-Layout-Auswahl oben verfügbar ist.</span><span class="sxs-lookup"><span data-stu-id="47aee-116">The** Report Layout Selection** window lists all the reports that are available in the company that is specified in the Company field at the top of the window.</span></span>
2. <span data-ttu-id="47aee-117">Legen Sie das Feld **Mandant** für den Mandanten fest, für den Sie das Berichtlayout erstellen möchten.</span><span class="sxs-lookup"><span data-stu-id="47aee-117">Set the **Company** field to the company in which you want to create the report layout.</span></span>
3. <span data-ttu-id="47aee-118">Wählen Sie im Fenster **Kundenspezifisches Layout** die Zeile für das benutzerdefinierte Layout, das Sie verwenden möchten, und wählen Sie dann die Schaltfläche OK aus.</span><span class="sxs-lookup"><span data-stu-id="47aee-118">Select the row for the report that you want to create the layout for, and then choose **Custom Layouts**.</span></span>  
   <span data-ttu-id="47aee-119">Das Fenster **Benutzerdefiniertes Berichtslayout** erscheint und alle benutzerdefinierten Layouts, die für den ausgewählten Bericht verfügbar sind, werden angezeigt.</span><span class="sxs-lookup"><span data-stu-id="47aee-119">The **Custom Report Layouts** window appears and lists all the custom layouts that are available for the selected report.</span></span>
4. <span data-ttu-id="47aee-120">Wenn Sie eine Kopie eines vorhandenen benutzerdefinierten Layouts erstellen möchten, wählen Sie das vorhandene benutzerdefinierte Layout in der Liste aus, und dann wählen **Kopieren**.</span><span class="sxs-lookup"><span data-stu-id="47aee-120">If you want to create a copy of an existing custom layout, select the existing custom layout in the list, and then choose **Copy**.</span></span>  
   <span data-ttu-id="47aee-121">Die Kopie des benutzerdefinierten Layouts erscheint im Fenster **Benutzerdefiniertes Layout** und hat den Begriff im Beschreibungsfeld.</span><span class="sxs-lookup"><span data-stu-id="47aee-121">The copy of the custom layout appears in the **Custom Report Layouts** window and has the words Copy of in the Description field.</span></span>

   1. <span data-ttu-id="47aee-123">Wählen Sie **Neu** aus.</span><span class="sxs-lookup"><span data-stu-id="47aee-123">Choose **New**.</span></span> <span data-ttu-id="47aee-124">Das Fenster **Integriertes Layout für einen Bericht** erscheint.</span><span class="sxs-lookup"><span data-stu-id="47aee-124">The **Insert Built-in Layout for a Report** window appears.</span></span> <span data-ttu-id="47aee-125">Die Felder **ID** und **Name** werden automatisch ausgefüllt.</span><span class="sxs-lookup"><span data-stu-id="47aee-125">The **ID** and **Name** fields are automatically filled in.</span></span>
   2. <span data-ttu-id="47aee-126">Um einen benutzerdefinierten WordBerichtlayouttyp hinzuzufügen, aktivieren Sie das **Word-Layout einfügen**-Kontrollkästchen.</span><span class="sxs-lookup"><span data-stu-id="47aee-126">To add a custom Word report layout type, then select the **Insert Word Layout** check box.</span></span>
   3. <span data-ttu-id="47aee-127">Um einen benutzerdefinierten RDLC-Berichtlayouttyp hinzuzufügen, aktivieren Sie das **RDLC-Layout einfügen**-Kontrollkästchen.</span><span class="sxs-lookup"><span data-stu-id="47aee-127">To add a custom RDLC report layout type, then select the **Insert RDLC Layout** check box.</span></span>
   4. <span data-ttu-id="47aee-128">Wählen Sie die Schaltfläche **OK** aus.</span><span class="sxs-lookup"><span data-stu-id="47aee-128">Choose the **OK** button.</span></span>  
      <span data-ttu-id="47aee-129">Die neuen benutzerdefinierten Layouts werden im Fenster **Benutzerdefinierte Berichtlayout** angezeigt.</span><span class="sxs-lookup"><span data-stu-id="47aee-129">The new custom layouts appear in the **Custom Report Layouts** window.</span></span> <span data-ttu-id="47aee-130">Wenn ein neues Layout auf einem integrierten Layout basiert, erscheinen die Begriffe **Kopie eines integrierten Layouts** im Feld **Beschreibung**.</span><span class="sxs-lookup"><span data-stu-id="47aee-130">If a new layout is based on a built-in layout, then it has the words **Copy of a Built-in Layout** in the **Description** field.</span></span> <span data-ttu-id="47aee-131">Wenn kein integriertes Layout für den Bericht vorhanden war, erhält das neue Layout die Begriffe **Neues Layout** im Feld **Beschreibung**, was angibt, dass das benutzerdefinierte Layout leer ist.</span><span class="sxs-lookup"><span data-stu-id="47aee-131">If there was no built-in layout for the report, then the new layout has the words **New Layout** in the **Description** field, which indicates that custom layout is blank.</span></span>
6. <span data-ttu-id="47aee-132">Standardmäßig ist das Feld **Unternehmensname** leer, d. h., dass das benutzerdefiniertes Layout für den Bericht in allen Unternehmen ist verfügbar.</span><span class="sxs-lookup"><span data-stu-id="47aee-132">By default, the **Company Name** field is blank, which means that the custom layout will be available for the report in all companies.</span></span> <span data-ttu-id="47aee-133">Um das benutzerdefinierte Layout nur in einem bestimmten Mandanten zu ändern, wählen Sie die Registerkarte **Bearbeiten**, und geben Sie dann im Feld **Unternehmensnamen** den Namen für den gewünschten Mandanten ein.</span><span class="sxs-lookup"><span data-stu-id="47aee-133">To make the custom layout available in a specific company only, choose **Edit**, and then set the **Company Name** field to the company that you want.</span></span>
