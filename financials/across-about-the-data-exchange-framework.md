# <a name="about-the-data-exchange-framework-in-included365finincludesd365finmdmd"></a>Über das Datenaustauschframework in [!INCLUDE[d365fin](includes/d365fin_md.md)]
Das Format von Dateien zum Austausch von Daten in den Bankdateien, die elektronischen Belegen, die Währungsumrechnungsraten und anderen mit ERP-Systemen variieren abhängig vom Anbieter der Datendatei oder des Streams und dem Land/der Region. [!INCLUDE[d365fin](includes/d365fin_md.md)] unterstützt verschiedene Bankdateiformate und Datendienststandards. Um Unterstützung für andere elektronische Belegformate zu bieten, verwenden Sie das Datenaustauschframework. Weitere Informationen finden Sie unter [Daten als elektronische Belege austauschen](across-data-exchange.md).
