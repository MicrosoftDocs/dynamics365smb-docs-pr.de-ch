<span data-ttu-id="45bd8-107">Mithilfe eines externen OCR-Dienstes (optische Zeichenerkennung) können Sie aus PDF- oder Bilddateien, die die Eingangsbelege darstellen, elektronische Belege generieren, die dann in Project '[!INCLUDE[d365fin](includes/d365fin_md.md)]' in Belegdatensätze konvertiert werden können.</span><span class="sxs-lookup"><span data-stu-id="45bd8-107">From PDF or image files representing incoming documents, you can have an external OCR service (Optical Character Recognition) generate electronic documents that can then be converted to document records inside [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span>

<span data-ttu-id="45bd8-108">Im Fenster **Eingehende Belege** können Sie verschiedene Funktionen zum Überprüfen von Ausgabenbelegen, Verwalten von OCR-Aufgaben und Konvertieren eingehender Belege, manuell oder automatisch, in den entsprechenden Belegen oder Erfassungsjournalzeilen verwenden.</span><span class="sxs-lookup"><span data-stu-id="45bd8-108">In the **Incoming Documents** window, you can use different functions to review expense receipts, manage OCR tasks, and convert incoming document files, manually or automatically, to the relevant documents or journal lines.</span></span> <span data-ttu-id="45bd8-109">Die externen Dateien können jeder Prozessphase zugeordnet werden, auch gebuchten Belegen und den resultierenden Kreditoren-, Debitoren- und Fibuposten.</span><span class="sxs-lookup"><span data-stu-id="45bd8-109">The external files can be attached at any process stage, including to posted documents and to the resulting vendor, customer, and general ledger entries.</span></span>

<span data-ttu-id="45bd8-110">Die Verarbeitung von Eingangsbelegen kann aus den folgenden Hauptaktivitäten bestehen:</span><span class="sxs-lookup"><span data-stu-id="45bd8-110">The incoming document process can consist of the following main activities:</span></span>

* <span data-ttu-id="45bd8-111">Erfassen Sie die externen Belege in Project '[!INCLUDE[d365fin](includes/d365fin_md.md)]', indem Sie mithilfe einer der folgenden Methoden Zeilen im Fenster **Eingehende Belege** anlegen:</span><span class="sxs-lookup"><span data-stu-id="45bd8-111">Record the external documents inside [!INCLUDE[d365fin](includes/d365fin_md.md)] by creating lines in the **Incoming Documents** window in either of the following ways:</span></span>
  * <span data-ttu-id="45bd8-112">Manuell, mithilfe von einfachen Funktionen, entweder von einem PC oder von einem mobilen Gerät aus, auf eine der folgenden Arten:</span><span class="sxs-lookup"><span data-stu-id="45bd8-112">Manually, by using simple functions, either from a PC or from a mobile device, in one of the following ways:</span></span>
    * <span data-ttu-id="45bd8-113">Verwenden Sie die Schaltfläche **Aus Datei erstellen** und füllen Sie dann die entsprechenden Felder im Fenster **Eingehender Beleg** aus.</span><span class="sxs-lookup"><span data-stu-id="45bd8-113">Use the **Create from File** button, and then fill relevant fields in the **Incoming Document** window.</span></span> <span data-ttu-id="45bd8-114">Die Datei wird automatisch angehängt.</span><span class="sxs-lookup"><span data-stu-id="45bd8-114">The file is automatically attached.</span></span>  
    * <span data-ttu-id="45bd8-115">Verwenden Sie die Schaltfläche **Neu**, füllen Sie dann die entsprechende Felder im Fenster **Eingehender Beleg** aus und hängen Sie die betreffende Datei an.</span><span class="sxs-lookup"><span data-stu-id="45bd8-115">Use the **New** button, and then fill relevant fields in the **Incoming Document** window and manually attach the related file.</span></span>
    * <span data-ttu-id="45bd8-116">Auf einem Tablet oder einem Telefon verwenden Sie die Schaltfläche **Von Kamera erstellen**, um einen neuen Eingangsbelegsdatensatz zu erstellen, und senden das Bild dann beispielsweise an den OCR-Dienst.</span><span class="sxs-lookup"><span data-stu-id="45bd8-116">From a tablet or phone, use the **Create from Camera** button to create a new incoming document record, and then send the image to the OCR service, for example.</span></span>
  * <span data-ttu-id="45bd8-117">Automatisch, indem Sie den Beleg vom OCR-Dienst als ein elektronisches Beleg empfangen, nachdem Sie die zugehörige PDF- oder Bilddatei per E-Mail an den OCR-Dienst gesendet haben.</span><span class="sxs-lookup"><span data-stu-id="45bd8-117">Automatically, by receiving the document from the OCR service as an electronic document after you have emailed the related PDF or image file to the OCR service.</span></span> <span data-ttu-id="45bd8-118">Das Inforegister **Finanzinformationen** wird automatisch im Fenster **Eingehender Beleg** ausgefüllt.</span><span class="sxs-lookup"><span data-stu-id="45bd8-118">The **Financial Information** FastTab is automatically filled in the **Incoming Document** window.</span></span>
* <span data-ttu-id="45bd8-119">Nutzen Sie den OCR-Dienst zur Umwandlung von PDF- oder Bilddateien in elektronische Belege die in [!INCLUDE[d365fin](includes/d365fin_md.md)] in Belegdatensätze konvertiert werden können.</span><span class="sxs-lookup"><span data-stu-id="45bd8-119">Use the OCR service to have PDF or image files turned into electronic documents that can be converted to document records in [!INCLUDE[d365fin](includes/d365fin_md.md)].</span></span>
* <span data-ttu-id="45bd8-120">Erstellen Sie neue Belege oder eine Fibu Buch.-Blattzeilen aus einem eingehenden Belegdatensatz, indem sie die Informationen so eingeben, wie sie in den eingehenden Belegen stehen.</span><span class="sxs-lookup"><span data-stu-id="45bd8-120">Create new documents or general journal lines for incoming document records by entering the information as you read it from incoming document files.</span></span>
* <span data-ttu-id="45bd8-121">Fügen Sie eingehender Belege zu Einkaufs- und Verkaufsbelegen jeden möglichen Status hinzu, einschliesslich der resultierenden Kreditor, Debitor- und Fibuposten, die aus dem Buchen resultieren.</span><span class="sxs-lookup"><span data-stu-id="45bd8-121">Attach incoming document files to purchase and sales documents of any status, including to the vendor, customer, and general ledger entries that result from posting.</span></span>
* <span data-ttu-id="45bd8-122">Zeigen Sie Eingangsbelege und deren Anhänge aus Einkaufs- und Verkaufsbelegen oder Posten an, oder finden Sie alle Sachposten ohne Eingangsbelege im Fenster **Kontenplan**.</span><span class="sxs-lookup"><span data-stu-id="45bd8-122">View incoming document records and their attachments from any purchase and sales document or entry, or find all general ledger entries without incoming document records from the **Chart of Accounts** window.</span></span>
