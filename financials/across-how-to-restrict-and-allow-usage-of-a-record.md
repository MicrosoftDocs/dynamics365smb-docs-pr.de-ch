>  <span data-ttu-id="deadc-109">Die generische Version von [!INCLUDE[d365fin](includes/d365fin_md.md)] unterstützt die Beschränkung der Buchung, des Exports als Zahlung und des Druckens als Scheck für einen Datensatz.</span><span class="sxs-lookup"><span data-stu-id="deadc-109">The generic version of [!INCLUDE[d365fin](includes/d365fin_md.md)] offers support for restricting a record from being posted, from being exported as a payment, and from being printed as a check.</span></span> <span data-ttu-id="deadc-110">Um andere Einschränkungen zu unterstützen muss ein Microsoft-Partner den Anwendungscode anpassen.</span><span class="sxs-lookup"><span data-stu-id="deadc-110">To support other restrictions, a Microsoft partner must customize the application code.</span></span>

## <a name="to-create-a-workflow-step-that-restricts-posting-of-unapproved-purchase-orders"></a><span data-ttu-id="deadc-114">So erstellen Sie einen Workflowschritt, der die Buchung von nicht genehmigten Einkaufsbestellungen einschränkt</span><span class="sxs-lookup"><span data-stu-id="deadc-114">To create a workflow step that restricts posting of unapproved purchase orders</span></span>  
1. <span data-ttu-id="deadc-115">Wählen Sie das Symbol ![Nach Seite oder Bericht suchen](media/ui-search/search_small.png "Symbol Nach Seite oder Bericht suchen"), geben **Workflows** ein und wählen dann den zugehörigen Link aus.</span><span class="sxs-lookup"><span data-stu-id="deadc-115">Choose the ![Search for Page or Report](media/ui-search/search_small.png "Search for Page or Report icon") icon, enter **Workflows**, and then choose the related link.</span></span>  
2. <span data-ttu-id="deadc-116">Im Fenster **Workflo**erstellen Sie einen neuen Workflow mit dem Namen "Einkaufsbestellungs-Genehmigungsworkflow".</span><span class="sxs-lookup"><span data-stu-id="deadc-116">In the **Workflows** window, create a new workflow named Purchase Order Approval Workflow.</span></span> <span data-ttu-id="deadc-117">Weitere Informationen finden Sie unter [Gewusst wie: Workflows erstellen](across-how-to-create-workflows.md).</span><span class="sxs-lookup"><span data-stu-id="deadc-117">For more information, see [How to: Create Workflows](across-how-to-create-workflows.md).</span></span>  
3. <span data-ttu-id="deadc-118">Wählen Sie die Aktion **Von Workflowvorlage kopieren** aus.</span><span class="sxs-lookup"><span data-stu-id="deadc-118">Choose the **Copy From Workflow Template** action.</span></span>  
4. <span data-ttu-id="deadc-119">Wählen Sie das **Quellworkflow-Code**-Feld aus und wählen Sie dann im Fenster  die **Workflowvorlage** "Einkaufsrechnungs-Genehmigungsworkflow" aus.</span><span class="sxs-lookup"><span data-stu-id="deadc-119">Choose the **Source Workflow Code** field, and then, in the **Workflow Templates** window, choose the Purchase Invoice Approval Workflow workflow template.</span></span>  

     <span data-ttu-id="deadc-120">Beachten Sie, dass die ersten beiden Workflowschritte zur Einschränkung und Zulassung der Nutzung von Einkaufsrechnungen dienen.</span><span class="sxs-lookup"><span data-stu-id="deadc-120">Notice that the first two workflow steps are about restricting and then allowing usage of purchase invoices.</span></span> <span data-ttu-id="deadc-121">Bearbeiten Sie die Ereignisbedingung des ersten Schritts des neuen Workflows. Legen Sie fest, dass er auf Einkaufsbestellungen angewandt wird.</span><span class="sxs-lookup"><span data-stu-id="deadc-121">Proceed to change the event condition on the first step of the new workflow to specify that it applies to purchase orders.</span></span>  
5. <span data-ttu-id="deadc-122">Wählen Sie im **Workflow-Schritte**-Inforegister das Feld **Ereigniskonditionen** , und wählen Sie dann für den Filter **Dokumentart** die Option **Auftrag** aus.</span><span class="sxs-lookup"><span data-stu-id="deadc-122">On the **Workflow Steps** FastTab, choose the **Event Conditions** field, and then, for the **Document Type** filter, select **Order**.</span></span>  
6. <span data-ttu-id="deadc-123">Löschen, bearbeiten oder erstellen Sie andere Workflowschritte, um einen Geschäftsprozess abzudecken, der mit der Einschränkung der Buchung von nicht genehmigten Einkaufsbestellungen beginnt.</span><span class="sxs-lookup"><span data-stu-id="deadc-123">Proceed to edit, delete, or add other workflow steps to fit a business process that begins by restricting unapproved purchase orders from being posted.</span></span>
